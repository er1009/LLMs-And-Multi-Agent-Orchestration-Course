# Assets Directory

This directory contains project assets including diagrams, images, and visual resources.

## Contents

### Architecture Diagrams
- Generated on-the-fly during execution
- System architecture visualizations
- Data flow diagrams

### Generated Graphs
All analysis graphs are saved in `results/graphs/`

### Documentation Images
Supporting images for documentation are stored here.

## File Types
- `.png` - Graph outputs, diagrams
- `.svg` - Vector graphics (if needed)
- `.md` - Asset documentation
